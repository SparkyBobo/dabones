# MIDI Channels

## Overview

The DaBones use MIDI channels to target individual song elements, with one MIDI channels for each "instrument", i.e., character or song element. The song elements are:

Channel|Element
-----|-----
0|Master/All channel
1|Joey, male singer
2|Lizzy, female singer
3|Witches, chorus singers
4|Beat, drums and percussion
5|Groove, bass guitar
6|Slash, electric guitar
7|Tickles, piano/keyboard
8-12|Reserved
13|Lights
14|Special Effects, e.g., strobe, mirror ball, smoke
15|Video playback
